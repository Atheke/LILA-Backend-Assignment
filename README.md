# Multiplayer Tic-Tac-Toe with Nakama

Backend-focused assignment implementation for a server-authoritative multiplayer Tic-Tac-Toe game.

## Project Structure

- `nakama/` - Nakama runtime modules and TypeScript build configuration
- `frontend/` - React + Vite web frontend
- `docker-compose.yml` - Local Nakama + Postgres environment
